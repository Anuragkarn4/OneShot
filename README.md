# OneShot
OneShot is a simple Unity 3D bow-and-arrow arcade game where the player aims and shoots arrows at falling fruits while avoiding bombs.
This project was created as part of my learning journey in Unity game development and C# scripting. The goal of the game is to test accuracy, timing, and reaction speed using a single bow mechanic.

## Gameplay
- Click and hold near the bow to start aiming
- Move your mouse to aim at the falling fruits
- Release to fire — the longer you hold, the more power behind the shot
- Hit fruits: +10 points each
- Hit bombs: -8 points and counts as a bomb hit
- 5 bomb hits: game over

## Built With
- Unity 3D
- C#
- TextMeshPro
- Unity Physics System
- Unity UI System
